Metas Do Projeto
    Apresentar dados referentes a época de pandemia e pós pandemia retirados pincipalmente do Datasus (Tabnet).
    Apresentar gráficos que comparam diversos dados medicos nas esferas municipais, estaduais e federais.
Jornalisticamente Falando:
Com base na área de atuação da Rede Vanguarda, seria possível tabular dados das principais cidades do Vale do Paraíba, como São José dos Campos, Jacareí e Taubaté, o que permitiria a produção de algumas reportagens. Embora seja comum que os dados sejam mais difíceis de encontrar em nível regional, informações em âmbito estadual já seriam suficientes para dar origem a diversas matérias. Além disso, dados nacionais também poderiam ser utilizados, sendo encaminhados para jornais em todo o país.

Com a pandemia afetando severamente o Brasil, um país que possui um sistema universal de saúde, é provável que o cenário pós-pandemia acarrete em aumento significativo no atendimento, procedimentos, tratamentos e uso de medicamentos relacionados aos casos de covid longa.

 

No entanto, existem poucos estudos disponíveis para mensurar o impacto disso no Sistema Único de Saúde em diferentes níveis administrativos. Portanto, a análise aprofundada dos dados do Datasus (Tabnet) pode fornecer informações relevantes sobre a sobrecarga do sistema de saúde nas esferas municipais, estaduais e federais durante o período pós-pandemia.
